# MVP Spec — Task Tracker

## Objective

Build the smallest useful task tracking application for the AEHF greenfield pilot.

## In scope

- create a task
- list tasks
- mark a task as done
- in-memory storage

## Out of scope

- authentication
- persistence database
- categories or priorities
- due dates
- collaboration
- external integrations

## Success criteria

- a user can create a task
- a user can see created tasks
- a user can mark a task as done
- the code remains small and easy to review
- the repository memory remains aligned with implementation
